# 评分系统功能开发计划

## 背景
用户需要在黑客松竞赛平台中添加完整的评分功能系统，包括：
- **选手** - 提交作品、查看自己的评分和排名
- **专家** - 评审作品、打分、添加评语
- **管理员** - 管理评分维度、权重配置、查看所有评分
- **观众** - 查看排行榜、评分详情
- **AI自动评分模块** - 根据提交内容自动打分

## 角色权限设计

| 角色 | 功能 |
|------|------|
| 选手 | 提交作品、查看个人排名和评分反馈 |
| 专家 | 查看待评审作品、提交评分、查看已评分列表 |
| 管理员 | 管理评分维度权重、AI评分配置、最终排名确认 |
| 观众 | 查看公开排行榜、评分概览 |

## 评分配置

### 评分维度与权重
- **创新性 (30%)** - 项目的创新程度、独特性
- **技术难度 (25%)** - 技术实现的复杂度
- **实用性 (25%)** - 实际应用价值
- **商业价值 (20%)** - 商业化潜力

### 评分权重比例
- **AI评分**: 30%
- **专家评分**: 70%

## 实现方案

### 1. 类型定义扩展 (`src/types/index.ts`)

添加完整的评分相关类型：

```typescript
// 评分维度
export interface ScoreCriteria {
  id: string;
  name: string;
  description: string;
  maxScore: number;    // 最高分(100)
  weight: number;      // 权重(0-1)
}

// AI评分
export interface AIScore {
  submissionId: string;
  scores: CriteriaScore[];
  totalScore: number;
  feedback: string;
  evaluatedAt: string;
}

// 专家评分
export interface ExpertScore {
  expertId: string;
  expertName: string;
  submissionId: string;
  scores: CriteriaScore[];
  totalScore: number;
  comment: string;
  evaluatedAt: string;
}

// 单项分数
export interface CriteriaScore {
  criteriaId: string;
  score: number;
  feedback: string;
}

// 综合评分记录
export interface ScoreRecord {
  submissionId: string;
  teamId: string;
  hackathonId: string;
  aiScore?: AIScore;
  expertScores: ExpertScore[];
  finalScore: number;
  rank?: number;
}

// 用户角色
export type UserRole = 'player' | 'expert' | 'admin' | 'viewer';

// 专家用户
export interface Expert extends User {
  role: 'expert';
  expertise: string[];  // 专业领域
  ratingCount: number;  // 已评分数量
}

// 管理员用户
export interface Admin extends User {
  role: 'admin';
}

// 作品提交状态
export type SubmissionStatus = 'draft' | 'submitted' | 'scoring' | 'scored';

// 评分配置
export interface ScoringConfig {
  aiWeight: number;           // AI评分权重 (0.3)
  expertWeight: number;       // 专家评分权重 (0.7)
  minExpertScores: number;     // 最少专家评分数量 (3)
  criteria: ScoreCriteria[];  // 评分维度配置
}
```

### 2. 评分服务 (`src/services/scoring.ts`)

**AIScoringService**
- 模拟AI评分逻辑（基于提交内容分析）
- 关键词匹配评估
- 返回各维度分数和反馈

**WeightedScoreCalculator**
- 合并AI和专家评分
- 计算加权平均分
- 更新排名

**LeaderboardCalculator**
- 按竞赛分组排名
- 计算排名变化
- 生成排行榜数据
